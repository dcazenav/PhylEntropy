Welcome to phylEntropy



What is phylEntropy

phylEntropy is a web-based data analysis and visualization application. The tool brings together several analysis modules representing different research themes: phylogeny/clustering, bio-statistics, measures of biological diversity, multivariate analyses, machine learning. The user can import his data as a CSV file before choosing a type of analysis to perform. The first column of our file contains the identifier of each “object”. The following columns contain the biological sequence (one character per column). The last columns (representing description variables) generally contain character strings (text).

Why phylEntropy

The phylEntropy application allows you to perform data analysis and visualization. Although there are already many tools for analyzing and visualizing data, our approach differs from other applications by allowing users to perform both classic bio-statistical analyzes and more specific analyzes that require usually the use of paid software or special experience. When we take the example of the representation of a minimum spanning tree containing annotations according to the data or characters studied, we often have to use paid software such as BioNumerics (http://www.applied-maths.com/bionumerics) to clearly represent data in terms of various parameters (or variables). The use of existing algorithms and JavaScript libraries makes it possible to offer the non-expert user a rapid analysis solution for their data. In addition, with the development of the phylEntropy application, we are trying to implement new analysis strategies by following a paradigm that differs from existing tools. For example, in this application, we consider “biological sequence” type objects that differ from classical DNA or protein sequences. This biological sequence is represented in the form of a string of characters that can contain numbers, letters, but also special characters (if it is a fairly complex or specific analysis).
